# Проект: Lead Webhook MVP

## Назначение

Учебный локальный сервис для приема заявок через webhook. Проект демонстрирует
базовые навыки работы с API, SQLite, валидацией, обработкой ошибок,
логированием и SMTP-уведомлениями.

## Цели

- Сохранять входящие заявки из `POST /lead` в SQLite.
- Возвращать понятные ошибки для некорректных запросов.
- Фиксировать события в UTF-8 логе.
- Отправлять email-уведомления через настройки из `.env`.
- Оставаться простым проектом начального уровня без лишних зависимостей.

## Ограничения

- Один endpoint и один основной сценарий: новая заявка.
- Локальный запуск без VPS.
- SQLite как единственная база данных.
- Дедупликация только по `name.lower()` и российскому телефону в формате
  `+7XXXXXXXXXX`.
