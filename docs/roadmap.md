# Roadmap

## Сейчас

- Локальный MVP для приема заявок через webhook.
- SQLite-хранилище.
- Простая дедупликация.
- SMTP-уведомление менеджеру.

## Возможные следующие шаги

- Добавить endpoint для просмотра последних заявок.
- Добавить статус обработки заявки.
- Добавить экспорт заявок в CSV.
- Добавить интеграцию с внешней CRM или таблицей.
- Вынести email-отправку в фоновую задачу, если появится нагрузка.
