# Changelog

## 2026-05-01

- Уточнена Windows PowerShell инструкция для `curl.exe`: JSON с кириллицей
  нужно передавать как UTF-8 bytes, иначе в SQLite могут попасть `?????`.
- Добавлена команда диагностики реальных байтов в SQLite через `hex(...)`.

## 2026-04-30

- Создана активная OpenSpec-спека `001-lead-webhook-mvp`.
- Добавлен FastAPI MVP для приема заявок через `POST /lead`.
- Добавлено хранение заявок в SQLite и дедупликация по имени и телефону.
- Добавлено UTF-8 логирование событий и SMTP-уведомление через `.env`.
- Добавлены ручные payload'ы и автотесты базового поведения.
- Добавлена документация для ручной проверки через `curl -s ... | jq`.
- JSON-ответы теперь явно отдаются с `charset=utf-8` для корректной кириллицы
  в Windows PowerShell.
